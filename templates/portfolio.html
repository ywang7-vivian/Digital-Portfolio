Boston Random Restaurant Website

Python HTML JavaScript CSS

The project aims to offer an alternative for people with decidophobia to find out what they want to eat every day in Boston.

Users can choose between "I am lucky today" (randomly choose one restaurant with type, location, and distance users enter) and "List all options" (Show all restaurants options that meet the conditions users enter).

The project is available online.

The source code can be found in github.
